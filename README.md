# Expense-Tracker-GUI
A simple and user-friendly Expense Tracker application built using Python and Tkinter. This project helps users to record, manage, and analyze daily expenses through a graphical user interface.


Features

 Add daily expenses with date, category, description & amount

 View all recorded expenses in one place

 Calculate total spending

 Category-wise expense summary

 Input validation with error handling

Clean & interactive GUI using Tkinte

echnologies Used

Python

Tkinter (GUI Framework)

MessageBox

List & Dictionary (Data Storage)
